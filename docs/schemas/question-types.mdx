# Question Types

Questions define the participant-facing inputs inside a survey section.

All question types share a common base structure:

- `id`
- `type`
- `text`
- `required`
- `hide_id`
- `hide_value`
- `hide_if`
- `rand_group`

Specific question types add their own required fields.
